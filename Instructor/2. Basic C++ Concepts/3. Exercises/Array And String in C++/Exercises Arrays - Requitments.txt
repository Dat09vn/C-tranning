Exercise 1
Create an array that can hold ten integers, and get input from user. Display those values on the screen, 
and then prompt the user for an integer. Search through the array, and count the number of times the item is found.

Exercise 2
Take 10 integer inputs from user and store them in an array and print them on screen.

Exercise 3
Take 20 integer inputs from user and print the following:
number of positive numbers
number of negative numbers
number of odd numbers
number of even numbers
number of 0.

Exercise 4
Take 10 integer inputs from user and store them in an array. Now, copy all the elements in another array but in reverse order.

Exercise 5
Find the largest and smallest elements of an array. Initialize default value: {23,6,328,34,12,234,9,23,4,54}
